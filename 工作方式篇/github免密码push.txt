window版本：
-先将你的项目从GitHub上clone下来
-需要生成秘钥
	-ssh-keygen -t rsa -C “171xxxx887@qq.com”
-打开秘钥文件，直接去C盘 c/user/xxx/.ssh目录下直接找，拷贝即可
	-cat ~/.ssh/id_rsa.pub 
-在github的setting / SSH and GPG keys中，新增ssh key，title任意可空，把id_rsa.pub中的东西复制到内容中就好。 
-输入以下指令即可：
	-git remote rm origin
	-git remote add origin git@github.com:***/BAT-algorithms.git
	-git push --set-upstream origin master
